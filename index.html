Past Projects

Take a look at my past and on-going programming projects!

Fingfind

➜

Fingfind is a pattern-matching program that finds biodata matches from a database using fingerprint images by utilizing the Knuth-Morris-Pratt (KMP) and Boyer-Moore (BM) algorithms. Fingfind was built with C# Programming Language and MySQL.

Belekping

➜

Belekping is a farming simulation game where users can buy, sell, and use cards featuring fields, foods, serums, and other characters. Users can grow crops by placing cards in the fields, save their current game state, resume previous game state, and collect points. Belekping was built with Java programming language.

WikiDiDi

➜

WikiDiDi is a locally built website that gathers all articles connecting two Wikipedia entries via hyperlinks, implementing the Wikirace game. WikiDiDi was built with Golang programming language supported by React and NodeJS.

ListLearn

➜

ListLearn is a local desktop application designed to help users improve time management, particularly for learning activities. ListLearn was built with Python programming language and SQLite.

Langkah Teman Kamu Web

➜

Langkah Teman Kamu website is a platform for the Langkah Teman Kamu organization to share important information, including monthly newsletters, ongoing and upcoming events, articles, organizational values, and more. The website is currently undergoing a redesign to provide an improved user interface and enhance the user experience for all visitors.
